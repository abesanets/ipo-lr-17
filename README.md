# Интернет-магазин электроники

## Описание

Учебный проект на Django, созданный в рамках лабораторных работ.
Приложение представляет собой основу интернет-магазина электроники
с моделями товаров, категорий, производителей и корзин пользователей.

## Модели

### Категория товара (Category)
- Название, описание

### Производитель (Manufacturer)
- Название, страна, описание

### Товар (Product)
- Название, описание, фото, цена, количество на складе
- Связь с категорией и производителем

### Корзина (Cart)
- Связь с пользователем (один к одному)
- Метод подсчёта общей стоимости

### Элемент корзины (CartItem)
- Связь с корзиной и товаром
- Количество, стоимость элемента
- Валидация по остатку на складе

## Запуск проекта

```bash
python -m venv venv
venv\Scripts\activate          # Windows
pip install django Pillow
python manage.py migrate
python manage.py createsuperuser
python fill_database.py
python manage.py runserver
```
